# Rotating Sphere of Interactive Circles

A visual experiment built with Processing.

The sketch displays a rotating 3D sphere made of moving circles.
The circles are simulated in 3D space but rendered as flat 2D shapes.

## Controls

+ / - → add or remove circles  
UP / DOWN → increase or decrease circle size  
LEFT / RIGHT → change rotation speed  

Mouse movement repels nearby circles.

## Technology

Built with the Processing creative coding framework.
